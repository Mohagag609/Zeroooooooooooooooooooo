# مدونة قواعد السلوك

## مبادئنا

نلتزم بإنشاء مجتمع شامل ومحترم للجميع.

## السلوك المتوقع

- كن محترماً ومهذباً
- استمع للآراء المختلفة
- ساعد الآخرين
- كن بنّاءً في النقد

## السلوك غير المقبول

- التحرش أو الإساءة
- التعليقات المسيئة
- التمييز
- السلوك العدواني

## الإجراءات

المخالفات ستؤدي إلى:
1. تحذير
2. تعليق مؤقت
3. حظر دائم

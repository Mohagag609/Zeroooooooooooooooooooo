# نظام إدارة المقاولات المحاسبي (ERP)

نظام إدارة شامل للمقاولات يتضمن إدارة المشاريع والمحاسبة والمخزون والموارد البشرية.

## 🚀 المميزات

### 📊 لوحة التحكم
- إحصائيات شاملة للأداء المالي
- رسوم بيانية تفاعلية للإيرادات والمصروفات
- تتبع تقدم المشاريع
- النشاطات الأخيرة والمهام القادمة

### 👥 إدارة العملاء والموردين
- إدارة شاملة لقاعدة بيانات العملاء
- تتبع الموردين والمشتريات
- إدارة الفواتير والمدفوعات
- تقارير الذمم المدينة والدائنة

### 🏗️ إدارة المشاريع
- إنشاء وإدارة المشاريع والمراحل
- تتبع التقدم والميزانيات
- إدارة الموارد والمواد
- تقارير تكلفة المشاريع

### 📦 إدارة المخزون
- إدارة المواد والمخازن
- تتبع حركات المواد
- تنبيهات المخزون المنخفض
- تقارير المخزون

### 💰 النظام المالي
- إدارة الإيرادات والمصروفات
- نظام الحسابات المحاسبية
- إدارة الخزائن والتحويلات
- التقارير المالية الشاملة

### 👨‍💼 إدارة الموارد البشرية
- إدارة الموظفين والمرتبات
- تتبع الإجازات والحضور
- إدارة الشركاء والتسويات

### 📈 التقارير
- ميزان المراجعة
- قائمة الدخل
- الميزانية العمومية
- تقارير التدفق النقدي
- تقارير أعمار الديون

## 🛠️ التقنيات المستخدمة

- **Frontend**: Next.js 14, React 18, TypeScript
- **UI Framework**: Tailwind CSS, Shadcn/ui
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Database**: Prisma ORM
- **Authentication**: NextAuth.js
- **Icons**: Lucide React

## 📦 التثبيت والتشغيل

### المتطلبات الأساسية
- Node.js 18+ 
- npm أو yarn
- قاعدة بيانات (PostgreSQL/MySQL)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd erp-system
```

2. **تثبيت التبعيات**
```bash
npm install
# أو
yarn install
```

3. **إعداد متغيرات البيئة**
```bash
cp .env.example .env.local
```

4. **تحديث ملف .env.local**
```env
DATABASE_URL="postgresql://username:password@localhost:5432/erp_db"
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"
```

5. **إعداد قاعدة البيانات**
```bash
npx prisma generate
npx prisma db push
```

6. **تشغيل المشروع**
```bash
npm run dev
# أو
yarn dev
```

7. **فتح المتصفح**
```
http://localhost:3000
```

│   ├── reports/           # التقارير
│   ├── settings/          # الإعدادات
│   └── notifications/     # الإشعارات
├── components/            # مكونات React
